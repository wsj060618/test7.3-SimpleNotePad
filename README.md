# 简易记事本 (SimpleNotePad)

一个使用Java Swing实现的简易记事本应用程序，提供基本的文本编辑和文件操作功能。

## 功能特性

### 文件操作
- 新建文件：清空当前编辑区并重置文件状态
- 打开文件：支持选择并打开本地文本文件
- 保存文件：保存当前编辑内容到已打开的文件
- 另存为：将当前内容保存为新文件
- 退出：关闭应用程序

### 编辑功能
- 撤销：清空当前编辑区内容

### 查看功能
- 自动换行：控制文本是否自动换行显示
- 字体放大/缩小：调整编辑区文本字体大小

## 快捷键支持
- `Ctrl+S`：保存文件
- `Ctrl+Z`：撤销操作
- `Ctrl+=`：字体放大
- `Ctrl+-`：字体缩小
- `ESC`：退出应用程序

## 技术实现
该项目基于Java Swing开发，使用了以下主要组件和技术：
- JFrame：应用程序主窗口
- JTextArea：多行文本编辑区域
- JMenuBar/JMenu/JMenuItem：菜单系统
- JScrollPane：滚动面板，支持大文本查看
- JFileChooser：文件选择对话框
- 事件监听器：处理用户交互操作
- NIO文件操作：读写文本文件

## 使用说明
1. 运行程序：执行`SimpleNotePad`类的`main`方法
2. 新建文件：点击"文件"菜单中的"新建"或使用菜单操作
3. 打开文件：点击"文件"菜单中的"打开"，选择要打开的文本文件
4. 保存文件：点击"文件"菜单中的"保存"或使用`Ctrl+S`快捷键
5. 调整字体大小：使用"查看"菜单中的"放大"/"缩小"选项或对应的快捷键
6. 自动换行：通过"查看"菜单中的"自动换行"选项控制

## 注意事项
- 撤销操作会清空整个编辑区内容
- 字体缩小有最小限制（8号字体），避免字体过小无法阅读
- 文件编码使用UTF-8，确保中文等特殊字符正确显示
